Introduction
============

Example package that shows how to extend the comment form. 

This currently works only with plone.app.discussion >= 1.1.2 (Plone 3/4) and plone.app.discussion >= 2.0.4 (Plone 4.1).

See the plone.app.discussion documentation for the howto:

http://packages.python.org/plone.app.discussion/howtos/howto_extend_the_comment_form.html
